# بازی آنلاین حکم (نسخه حرفه‌ای)

این پروژه یک بازی کامل چندنفره حکم با رابط فارسی، چت، لابی عمومی/خصوصی، سیستم امتیازدهی تیمی، لیدربرد و به‌روزرسانی لحظه‌ای با Polling است.

## ویژگی‌ها

- احراز هویت کامل: ثبت‌نام، ورود، خروج، هش رمز با `bcrypt`
- لابی عمومی: نمایش کاربران آنلاین و اتاق‌های عمومی
- اتاق خصوصی: لینک دعوت یکتا، عدم نمایش در لیست عمومی
- بازی حکم ۴ نفره تیمی (۲ به ۲)
- انتخاب حکم توسط دیلر
- رعایت پیروی از خال
- محاسبه برنده هر دست و امتیاز تیمی تا پایان بازی
- چت داخل اتاق با محدودیت ضداسپم
- لیدربرد بر اساس برد/باخت/امتیاز
- رابط کاربری دارک و حرفه‌ای با چیدمان واقعی میز حکم

## ساختار پروژه

- `public/` صفحات و API
- `assets/css/` استایل‌ها
- `assets/js/` منطق رابط کاربری
- `classes/` کلاس‌های OOP (خدمات)
- `config/` تنظیمات و بوت‌استرپ
- `sql/` اسکریپت دیتابیس

## نصب و اجرا

1. کد را در وب‌سرور (Apache/Nginx + PHP 8.1+) قرار دهید.
2. فایل `config/config.php` را با اطلاعات دیتابیس خود تنظیم کنید.
3. دیتابیس را بسازید:
   ```bash
   mysql -u root -p < sql/schema.sql
   ```
4. دایرکتوری `public` را به عنوان DocumentRoot معرفی کنید (یا با پیشوند `/public` اجرا کنید).
5. وارد `http://localhost/public` شوید.

## نکات فنی

- ارتباط دیتابیس با PDO و `prepared statements`
- معماری OOP با سرویس‌های مستقل
- Polling بهینه هر ۱.۵ ثانیه (با `revision` برای دریافت تغییرات)
- امنیت سشن و پاک‌سازی ورودی
